## 📦 What this is

Lateralus is a programming language for people who want to write and run code with a small, direct toolchain. It includes a compiler, a virtual machine, an OS layer, a language server, and a VS Code extension.

If you want to try it on your Windows PC, use the release page and get the latest build from there.

## 🪟 Install on Windows

1. Go to the release page.
2. Pick the newest version.
3. Download the Windows package.
4. If Windows asks for permission, choose Run or Keep, then allow the file to finish.
5. If the file is a ZIP archive, right-click it and choose Extract All.
6. Open the extracted folder.
7. Look for the main app file, such as an .exe file.
8. Double-click it to start Lateralus.

If Windows SmartScreen appears, choose More info, then Run anyway if you trust the file from the release page.

## 🧩 Main parts

### Compiler
The compiler turns your code into a form the runtime can use. This helps the language work with a clean pipeline.

### Virtual machine
The VM runs compiled output in a controlled way. This keeps the runtime separate from the editor and source code.

### OS layer
The OS work lets the language connect more closely to the system. This is useful for lower-level work and bare-metal style projects.

### Language server
The language server helps tools like VS Code understand your code. It can power hints, errors, and code navigation.

### VS Code extension
The extension gives editor support inside VS Code. It helps the language feel easier to use day to day.
